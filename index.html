System Dynamics and Control

What is step input?

Step input is an instanteneous increase of the input from 0 to the magnitude of 1. This magnitude stays constant as time goes to infinity

First Order System

A first order system has the following general form: \(\frac{dy}{dt} + ay = U(t)\), in which $$a = \frac{1}{\tau} $$ Thus the homogeneous solution is \(y= Ce^{-t/\tau}\)

Second Order System

When \(a \ne 0\), there are two solutions to \(ax^2 + bx + c = 0\) and they are $$x = {{b \pm \sqrt{b^2-4ac}}/{2a}}.$$
